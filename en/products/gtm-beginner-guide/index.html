The essentials first

3 concepts - and you understand GTM.

Everything in GTM revolves around Tags, Triggers and Variables. Understand these three and you understand GTM.

T
Tag - What happens?

A tag is the code that gets executed. Example: "Send a pageview to GA4." The tag itself is passive - it only does something when a trigger tells it to.

T
Trigger - When does it happen?

A trigger is the condition that fires a tag. Example: "Every time someone views a page." Or: "Every time someone clicks a specific button."

V
Variable - What is the value?

A variable is a placeholder for a value that changes. Example: {{Page URL}} = the current URL. {{Click Text}} = the text of the element just clicked.
